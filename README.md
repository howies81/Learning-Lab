## ⚽ The Current Project
I am currently working on a simple project called a **Sodium Scanner**. This tool is designed to use the camera feed from a phone to read the barcode and determine if the food item is high in sodium or not, according to the criteria of the Pan American Health Organisation (PAHO) **Nutrient Profile Model**. If the sodium content exceeds the PAHO criteria, a black octagonal warning label with the text **"High In Sodium"** will be displayed on the screen, with the sodium content displayed next to the warning label
