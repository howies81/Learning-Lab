# 📰 TT News Hub
A personal news aggregator dashboard built for Trinidad & Tobago, displaying headlines and links from major local media outlets in one place.
The Problem
Keeping up with local news in T&T means juggling multiple websites and Facebook pages, consuming time most people don't have. TT News Hub pulls everything into a single dashboard so you can scan the day's headlines in minutes, rather than bouncing between tabs.

## Features

- Aggregates RSS feeds from CNC3, TTT, Trinidad Express, and Wired868
- Automatically updates daily via GitHub Actions
- Clean card layout with direct links to full articles
- No ads, no paywalls, no noise

## Built With

- Python
- Streamlit
- GitHub Actions
